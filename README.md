# QR Generator

A simple and responsive QR Code Generator built with HTML, CSS, and JavaScript. This project allows users to enter text, choose a QR code size, generate the QR code instantly, and download it for later use.

## Live Demo

[View Project](https://nayan-dasauni.github.io/Qr_Generator/)

## Features

- Generate QR codes instantly from text or links.
- Choose QR code size from 100x100 to 1000x1000.
- Download the generated QR code.
- Simple and beginner-friendly user interface.

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```bash
Qr_Generator/
│── index.html
│── style.css
│── script.js
```

## How It Works

1. Enter any text or URL in the input field.
2. Select the QR code size.
3. Click the **Generate** button.
4. Click the **Download** button to save the QR code.

## Installation

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/nayan-dasauni/Qr_Generator.git
   ```

2. Open the project folder:
   ```bash
   cd Qr_Generator
   ```

3. Open `index.html` in your browser.
